# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful, assume good faith, and keep discussion focused on
the work. Harassment or personal attacks aren't welcome.

Report unacceptable behaviour privately to the maintainer via a
[GitHub Security Advisory](https://github.com/VijitSingh97/bitcoin-starter-stack/security/advisories/new)
or a direct message. Reports are handled confidentially.
